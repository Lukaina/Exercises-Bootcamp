W3 Schools

W3Schools es un sitio web para aprender tecnologías web en línea. Contiene tutoriales de HTML, CSS, JavaScript, SQL, PHP, XML y otras tecnologías.​ Fue lanzado en 1999 por la empresa noruega Refsnes Data, y su nombre proviene de la World Wide Web.

Stack Overflow

Stack Overflow (en inglés literalmente "desbordamiento de pila") es un sitio web desarrollado por Jeff Atwood utilizado por una comunidad de desarrolladores informáticos, en la cual otros desarrolladores pueden encontrar soluciones a problemas de programación en diferentes lenguajes.

Página web creada por Juan Camilo Bustamante Hernandez para el bootcamp de Pioneras Dev.
